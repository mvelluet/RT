# RT

## Usage

./rt <flags> <xml file>

#### available flags:

 -title <name>

 -size <width> <height>

 -device <cpu/gpu>

 -iterations <n>

 -antialias <antialiasing level>

 -filtre <n> n (1 (negative image) 2 (shades of Grey) 3 (sepia))

 -print_keys

 -angle <radians/degrees>
 
 -debug
